# Team Task Manager

A full-stack web app for managing projects and tasks with role-based access control.

## 🔗 Live Links
- **Frontend:** https://taskmanagerliveee.netlify.app
- **Backend API:** https://team-task-manager-production-b306.up.railway.app

## 🚀 Features
- Authentication (Signup/Login with JWT)
- Role-based access control (Admin/Member)
- Project creation and team management
- Task creation, assignment and status tracking
- Dashboard with task overview and overdue tasks

## 🛠️ Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | React + Vite + Tailwind CSS |
| Backend | Node.js + Express |
| Database | MySQL |
| Auth | JWT |
| Deployment | Railway (Backend) + Netlify (Frontend) |

## ⚙️ Local Setup

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
